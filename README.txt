Para gerar o arquivo SIR_Model.csv, deve-se preencher o arquivo "SIR_Model_parametros.csv" com os seguintes dados, respeitando os espaçamentos e as vírgulas:
S,I,R,h,Nb,Tb,Sb,Ib,mk,nk,Tk,t

Comando de compilar e rodar o programa no Linux:
make all
